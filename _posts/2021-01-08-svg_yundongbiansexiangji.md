### 运动中信号逐渐变强的警示三角

#### 添加颜色就会出现一个信号逐渐增强的警示三角啦，我的期末项目期限也快变成红色三角啦

<style>
.lovestar svg {
			    animation: rotate 5s infinite;
				width: 25rem;
				height: 12.5rem;
			}
			
			@keyframes rotate {
			    0% {
			        transform: translate(-100px,0px);
			        fill: blue;
			    }
			    25%{
			        transform: translate(100px,0px);
			        fill: yellow;
			    }
			    50%{
			        transform: translate(100px,200px);
			        fill: orange;
			    }
			    75%{
			        transform: translate(-100px,200px);
			        fill: red;
			    }
			    100% {
			        transform: translate(-100px,0px);

			    }
			}	
</style>
